## About the Project

This project is an interactive **Retail Sales Analytics Dashboard** built using **Microsoft Power BI**. 
The objective was to transform raw retail sales data into meaningful business insights through 
data cleaning, data modeling, DAX calculations, and interactive visualizations.

The dashboard helps analyze sales performance, profitability, customer segments, regional performance, product trends, 
and returned orders to support business decision-making.

## Dashboard Overview

The dashboard provides insights into:

- Total Sales
- Total Profit
- Profit Margin
- Return Rate
- Monthly Sales Trend
- Sales by Region
- Sales by Customer Segment
- Top 10 Products by Sales
- Sales vs Target
- Regional Sales Analysis (Drill-through)

The report contains two pages:

- **Global Retail Sales Overview**
- **Regional Sales Analysis**

## Data Modeling

The project follows a **Star Schema** consisting of:

- **Orders** (Fact Table)
- **Returns** (Dimension Table)
- **People** (Dimension Table)

Relationships were created using:

- Order ID
- Region

---

## DAX Measures

The following measures were created:

- Total Sales
- Total Profit
- Total Quantity
- Profit Margin %
- Returned Orders
- Return Rate %
- Average Shipping Cost
- Sales Target
